# Disease Prediction using Machine Learning

## Objective
This project predicts heart disease using machine learning algorithms.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm Used
- Random Forest Classifier

## Dataset
Heart Disease Dataset from Kaggle

## Features
- Data Preprocessing
- Data Visualization
- Model Training
- Accuracy Evaluation
- Disease Prediction

## Accuracy
Approximately 85% to 90%

## How to Run

```bash
python disease_prediction.py
```
